# 🏠 Real Estate End-to-End Data Analytics Project (SQL + Power BI)

## 📌 Project Overview

This project demonstrates a complete end-to-end data analytics workflow using a relational SQL database as the primary data source and Power BI for interactive business intelligence reporting.

The dashboard enables stakeholders to monitor sales performance, evaluate agent productivity, analyze client contribution, and understand geographic revenue distribution.

---

## 🎯 Business Objectives

* Track overall sales revenue and transaction volume
* Identify top-performing agents and clients
* Analyze city-wise revenue distribution
* Understand price segment contribution
* Monitor monthly revenue trends
* Enable detailed drill-through analysis for individual agents

---

## 🛠️ Tools & Technologies Used

* **SQL (MySQL)** – Database design, data storage, analytical queries
* **Power BI Desktop** – Dashboard development and visualization
* **DAX** – KPI calculations and analytical measures
* **Power Query** – Data transformation and modeling
* **Direct SQL Connection** – Live data integration

## 📈 Key Business Insights

* Bengaluru contributes the highest share of total revenue
* Mid-range property segment dominates overall sales
* Top agents generate a significant portion of company revenue
* Revenue trends show seasonal performance variation
* Certain cities have lower deal count but higher average sale price
